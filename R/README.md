# Functions

This folder is composed of all the exported and utility functions used in {stockplotr}. These utility functions are typically used to perform intermediary tasks in the process of making a figure or table, and are run within many table/figure-making functions.
Each script depicts the specific function withi the file. The utility file functions include

- [`utils_plot.R`]: utility functions to make plots
- [`utils_table.R`]: utility functions to make tables
- [`utils_rda.R`]: utility function to create/export rda files when exporting tables or plots
- [`utils`]: general helpful utility functions mostly used in the `convert_output` function
