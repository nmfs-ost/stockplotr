# Inst Folder

This folder contains various files which support the creation of materials for 
running functions in {stockplotr}.

## Sub-folders

* The `extdata` sub-folder contains an example of an SS3 report file that can be
input into `convert_output()`.
* The `resources` folder contains the .csv files which change the native labels
found in a model file input into `convert_output` and the standardized labels 
that gets used in the function. This sub-folder also contains a template for the
captions and alternative text as well as the key quantities.
